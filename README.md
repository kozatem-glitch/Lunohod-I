## Стек используемых технологий
Python 3 Используется как основной язык программирования, благодаря простоте синтаксиса и большому количеству библиотек для анализа, логирования и визуализации.

Библиотека kRPC Позволяет управлять ракетой в игре Kerbal Space Program в реальном времени. Через неё реализованы: считывание телеметрии; управление тягой двигателя; автопилот; отделение ступеней;тповороты ракеты; постановка на орбиту.

CSV-файл логирования, применяется для сохранения телеметрии и дальнейшего анализа траектории.

Библиотека NumPy
Используется для эффективной работы с числовыми массивами и математическими операциями. Обеспечивает высокую производительность при обработке телеметрических данных и расчётах орбитальных параметров.

Библиотека Matplotlib
Применяется для визуализации данных полёта: построения графиков высоты, скорости, и массы аппарата. Позволяет наглядно анализировать этапы миссии.

Модуль scipy.integrate
Используется для численного решения дифференциальных уравнений движения. С его помощью моделируется полёт, учитывающая гравитацию, сопротивление атмосферы и другие физические параметры Кербина.
